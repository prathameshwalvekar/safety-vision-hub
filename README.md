# AI Safety Vision Hub

A comprehensive AI-powered safety monitoring system with multiple detection capabilities including helmet detection, fall detection, fire/smoke detection, phone usage detection, restricted area monitoring, and more.

## Features

- **Helmet Detection**: Ensures safety compliance in industrial environments
- **Fall Detection**: Real-time monitoring for workplace safety
- **Fire & Smoke Detection**: Early warning system for fire hazards
- **Phone Usage Detection**: Monitors distracted behavior in restricted areas
- **Restricted Area Detection**: Ensures zone compliance
- **Line Crossing Detection**: Monitors boundary violations
- **No Parking Detection**: Automated parking enforcement
- **Sterile Zone Monitoring**: Maintains clean room compliance
- **Traffic Monitoring**: Vehicle and pedestrian flow analysis

## Technology Stack



- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Python, Flask, OpenCV, YOLO
- **AI Models**: YOLOv8, YOLOv11, Custom LSTM classifiers
- **Real-time Processing**: WebRTC, WebSocket connections

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Access the application at `http://localhost:5173`
